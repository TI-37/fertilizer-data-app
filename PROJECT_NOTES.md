# Project Notes

## Reusable UI Ideas

- NPK property selector strip: compact horizontal control with three small selectors labeled `N`, `P`, `K`. Useful later for a dedicated fertilizer-property editor or advanced component settings. Original options:
  - N: 未選択, アンモニア態窒素, 硝酸態窒素, 硝酸態窒素＋アンモニア態窒素, 窒素全量, 緩効性窒素, 被覆窒素
  - P: 未選択, 水溶性りん酸, 可溶性りん酸, く溶性りん酸, りん酸全量
  - K: 未選択, 水溶性加里, く溶性加里, 加里全量
